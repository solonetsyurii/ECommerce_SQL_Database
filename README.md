# Data Engineering & SQL Analytics Portfolio

Welcome to my portfolio repository. This project showcases my skills in relational database architecture, SQL query optimization, and backend automation.

## Project Structure

### 1. E-Commerce Relational Database
This folder contains the architecture for a clothing and footwear e-commerce platform.
* **Database Schema (ERD):** A robust relational model normalized to 3NF, consisting of 16 interconnected tables covering clients, products, orders, and logistics.
* **SQL Implementation:** The `Yurii_Solonets_71466.sql` file contains the full DDL scripts to reconstruct the database schema.
* **Analytical Queries:** The `Zapytania.sql` file includes 17 advanced T-SQL queries demonstrating complex data retrieval, including:
    * **Window Functions** for ranking and data analysis.
    * **PIVOT** operations for sales reporting.
    * **CTEs (Common Table Expressions)** for complex joins and data aggregation.

## Tech Stack
* **Database:** Microsoft SQL Server (T-SQL)
* **Design Tools:** Entity-Relationship Modeling (ERD)
